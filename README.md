A simple HTML & CSS project that displays personal bio-data in a clean, styled format. Ideal for beginners learning web development or creating a basic online CV/profile.

Features

Shows personal details: Name, Class, Stream, Age, Gender, Address, Contact, Hobby, Skills.

Clean, centered card design with borders, shadows, and rounded corners.

Styled using internal CSS with readable fonts and highlighted labels.

Usage

Clone or download the repository.

Open index.html in any web browser.

Update the information as needed.
